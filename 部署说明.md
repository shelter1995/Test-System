# AI 话术陪练系统 — 部署说明

## 项目概述

企业 AI 工作台，集成知识库管理、AI 内容生成、话术陪练。包含两个核心服务：

- **RAG 知识库服务**（端口 8003）：文档解析、向量/图谱检索
- **AI 话术陪练系统**（端口 8002）：场景会话、内容生成、统一工作台

## 环境要求

| 组件 | 最低要求 | 说明 |
|------|---------|------|
| Python | 3.12 ~ 3.13 | 3.14 兼容性不佳 |
| 内存 | ≥ 8 GB | MinerU 解析大 PDF 时占用高 |
| 磁盘 | ≥ 10 GB | MinerU 模型约 3-5 GB |
| 浏览器 | Chrome / Edge / Firefox | 现代浏览器 |

## 安装步骤

### 1. 获取项目

```bash
git clone <仓库地址>
cd Test-System
```

### 2. 创建虚拟环境

```bash
python -m venv .venv

# Windows
.venv\Scripts\activate

# Linux / macOS
source .venv/bin/activate
```

### 3. 安装依赖

```bash
# RAG 服务依赖
pip install -r rag-anything-api/requirements.txt

# Tutor 服务依赖
pip install -r ai-tutor-system/requirements_tutor.txt

# 可选：MinerU 文档解析（完整 PDF/Office 支持）
pip install -U "mineru[core]"
```

> RAG-Anything 引擎源码已内置在 `rag-anything-api/raganything/` 中，无需额外安装。

### 4. 配置环境变量

```bash
cp rag-anything-api/.env.example rag-anything-api/.env
cp ai-tutor-system/.env.example ai-tutor-system/.env
```

编辑 `rag-anything-api/.env`：

```ini
# 必填
MINIMAX_API_KEY=你的MiniMax_API_Key
SILICONFLOW_API_KEY=你的硅基流动_API_Key

# 可选
ENABLE_VLM=true
ENABLE_RERANK=true
DEFAULT_QUERY_MODE=hybrid
```

编辑 `ai-tutor-system/.env`：

```ini
MINIMAX_API_KEY=你的MiniMax_API_Key
MINIMAX_MODEL=MiniMax-M2.7
RAG_SERVICE_URL=http://localhost:8003
```

密钥获取：
- MiniMax：https://platform.minimaxi.com
- 硅基流动：https://siliconflow.cn

### 5. 启动服务

**终端 1 — RAG 服务：**

```bash
cd rag-anything-api
python start.py
```

**终端 2 — Tutor 服务：**

```bash
cd ai-tutor-system
python tutor_backend.py
```

### 6. 验证部署

```bash
# RAG 健康检查
curl -s http://localhost:8003/health
# 期望：{"status":"healthy","engine":"ready"}

# Tutor 服务检查
curl -s http://localhost:8002/
```

浏览器访问：**http://localhost:8002**

## 生产部署

### 配置防火墙

开放端口：
- 8003（RAG 服务）
- 8002（Tutor 服务）

### 使用进程管理器

```bash
# pm2 示例（需先安装 Node.js）
npm install -g pm2

cd rag-anything-api
pm2 start "python start.py" --name rag-service

cd ../ai-tutor-system
pm2 start "python tutor_backend.py" --name tutor-service
```

### Nginx 反向代理示例

```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://localhost:8002;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    location /rag/ {
        proxy_pass http://localhost:8003/;
        proxy_set_header Host $host;
    }
}
```

## 常见问题

### 端口被占用

修改 `.env` 中的端口配置，或关闭占用端口的程序。

### 依赖安装失败

```bash
python -m pip install --upgrade pip
pip install package_name
```

### 模型加载失败

确认模型文件完整，检查模型路径配置。

### MiniMax API 错误

检查 API 密钥配置，确保余额充足。

### MinerU 安装失败

MinerU 为可选依赖。跳过安装后，知识库仍可通过文本导入正常工作。
