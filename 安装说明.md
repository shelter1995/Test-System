# 智学工作台 安装说明

> AI 销售话术陪练与 RAG 知识库系统 · Windows 桌面版安装手册

本手册面向 **使用 EXE 安装包部署的终端用户**。如果你是开发者，需要从源码部署，请参考 [部署说明.md](部署说明.md)。

---

## 1. 产品简介

智学工作台是一个**完全本地化运行**的 AI 销售训练系统，包含两个核心能力：

- **RAG 知识库**：把产品资料、话术文档、案例 PDF/Word/图片/音视频导入成可检索的知识库
- **AI 话术陪练**：选择客户场景，与 AI 进行多轮模拟对练，每轮自动评分

所有数据存在你自己的电脑上，不上云。

---

## 2. 系统要求

| 项目 | 最低配置 | 推荐配置 |
|------|---------|---------|
| 操作系统 | Windows 10 64位 (1809+) | Windows 11 64位 |
| 内存 | 8 GB | 16 GB（含视频转写时建议） |
| 可用磁盘 | 10 GB | 20 GB（含 Whisper/MinerU 模型） |
| 网络 | 需要 | 需要 |
| WebView2 Runtime | 随安装包自动部署 | 通常无需手动处理 |
| 屏幕分辨率 | 1366 × 768 | 1920 × 1080 |

> **说明**：本软件不需要 Python、Node.js、conda 等开发环境，安装包自带运行时。

## 3. 安装步骤

### 3.1 下载安装包

从内部共享或官方渠道获取 `智学工作台-Setup-x.x.x-x64.exe`（约 290 MB）。

### 3.2 运行安装包

1. **双击** `智学工作台-Setup-x.x.x-x64.exe`
2. 如果弹出"用户账户控制"（UAC）提示，选择"是"
3. 安装向导启动，等待约 10–30 秒

### 3.3 选择安装位置

- **默认**：`C:\Users\<你的用户名>\AppData\Local\Programs\智学工作台`
- 如需改路径，点击"浏览"选择其他位置（建议保持默认）

> 安装位置只放程序文件，所有**数据**会放在下一步设置的数据目录里，两者解耦便于升级与迁移。

### 3.4 选择数据目录

- **默认**：`C:\Users\<你的用户名>\AppData\Local\智学工作台\Data`
- 数据目录存放：知识库文件、转写产物、模型缓存、日志
- **建议**：保持默认，不要放在 C 盘根目录或 U 盘
- 升级安装时会自动沿用之前的数据目录，**无需重新导入知识库**

### 3.5 等待安装完成

安装过程中会：
1. 解压程序文件
2. 自动安装 Microsoft Edge WebView2 Runtime（如缺失）
3. 写入注册表与开始菜单快捷方式

### 3.6 安装结束

安装完成后，向导会提示两个可选动作（**默认不勾选**）：

- ☑️ **启动 智学工作台**：立即打开主程序
- ☑️ **安装 MinerU 增强解析组件**：处理扫描版 PDF/图片/音视频

> 第二项**强烈建议勾选**（详见下一节）。

## 4. 安装增强解析组件

### 4.1 什么是"增强解析组件"？

智学工作台默认只打包**基础解析能力**（文本、简单 PDF、现代 Office）。以下高级场景需要先安装增强解析组件：

- 📄 扫描版 PDF（图片型 PDF）
- 🖼️ 图片 OCR（识别图片中的文字）
- 🎬 视频/音频语音转写（用 Whisper 把说话内容转成文字）
- 📊 老格式 Office（`.doc`、`.xls`、`.ppt`）

> 第一次安装约需下载 1.5–3 GB（包含 MinerU 模型 + ffmpeg + Whisper 模型），**请保持网络畅通**。

### 4.2 安装方式

**方式一：安装结束时勾选**（推荐）
安装向导最后一页勾选"安装 MinerU 增强解析组件"，程序会自动弹出独立安装窗口。

**方式二：从主程序内安装**
1. 启动智学工作台
2. 左上角菜单 → **安装/修复增强解析组件**
3. 选择模型源（默认无需选择）
4. 点击"开始安装"，等待 5–20 分钟
5. 出现"增强解析组件安装完成"即可关闭窗口

## 5. 启动应用

### 5.1 首次启动

1. 双击桌面"智学工作台"图标，或开始菜单搜索
2. 首次启动会**自动初始化**两个本地服务（端口 8002、8003），约 5–15 秒
3. 内嵌浏览器自动打开主界面

### 5.2 看到"启动失败"怎么办？

最常见原因是**端口被占用**（8002/8003）。请参考 [使用说明.md → 常见问题 FAQ](使用说明.md#常见问题-faq) 第 5 条。

### 5.3 关闭应用

- 直接关闭主程序窗口即可，两个本地服务会**自动停止**
- **不要**用任务管理器强杀 `TestSystem.exe`，可能导致下次启动异常

---

## 6. 升级

### 6.1 升级流程

1. 下载新版本 `智学工作台-Setup-x.x.x-x64.exe`
2. 双击运行（**无需先卸载旧版**）
3. 安装向导检测到已安装的版本，会自动保留：
   - 数据目录（所有知识库、转写产物、配置）
   - 已下载的模型（MinerU/Whisper）
4. 等待升级完成

### 6.2 注意事项

- **不要在升级过程中关闭电脑或拔掉电源**
- 升级前主程序必须**先关闭**
- 如果升级后启动异常，参考"获取日志"章节，把日志发给开发人员

---

## 7. 卸载

### 7.1 通过控制面板卸载

```
设置 → 应用 → 已安装的应用 → 智学工作台 → 卸载
```

### 7.2 卸载选项

卸载向导会询问：
- **同时删除数据目录**（知识库、转写产物、模型、配置）— 不可恢复，**请谨慎勾选**
- **仅卸载程序**（保留数据，下次重装可继续使用）

### 7.3 完全清理

如果想彻底删除智学工作台的所有痕迹（包含缓存、注册表）：

1. 卸载主程序（取消勾选"保留数据"）
2. 手动删除数据目录：`C:\Users\<用户名>\AppData\Local\智学工作台`
3. 删除注册表项：`HKEY_CURRENT_USER\Software\ZhiXueWorkbench`

## 8. 数据目录说明

### 8.1 目录结构

```
%LOCALAPPDATA%\智学工作台\Data\
├── config\                  # 配置文件（install.json, .env）
├── logs\                    # 日志
│   ├── desktop-host.log     # 桌面宿主日志
│   ├── rag-service.log      # 知识库服务日志
│   ├── tutor-service.log    # 陪练服务日志
│   └── mineru-installer.log # 增强解析组件安装日志
├── models\                  # 模型缓存
│   ├── mineru\              # MinerU 模型
│   └── whisper\             # Whisper 模型
├── rag\                     # 知识库存储
│   ├── storage\             # 向量库 + 元数据
│   └── output\              # 解析产物（Markdown/分段）
├── tutor_data\              # 陪练会话历史
├── generation_output\       # 内容生成产物
└── runtime\                 # 运行时
    ├── optional-site-packages\  # 增强解析依赖（pip 装在这里）
    └── webview2-user-data\      # WebView2 用户数据
```

### 8.2 迁移到新电脑

1. 在旧电脑关闭智学工作台
2. 复制整个数据目录到新电脑相同位置（或通过安装向导选同样路径）
3. 新电脑安装同版本或更高版本智学工作台
4. 启动后会自动加载原数据

### 8.3 备份

建议定期备份 `config\`、`rag\storage\`、`tutor_data\` 这三个目录。模型可以重新下载，但**知识库原文与会话记录一旦丢失无法重建**。

## 9. 防火墙与安全软件

部分杀软/防火墙会拦截 Python 进程访问网络或读写磁盘，可能导致：

- 安装增强解析组件时下载失败
- 知识库保存失败
- 启动缓慢

如出现上述问题，请在杀软白名单中加入：

```
<安装目录>\runtime\python\python.exe
<安装目录>\TestSystem.exe
<数据目录>\runtime\optional-site-packages\  (整目录)
```
